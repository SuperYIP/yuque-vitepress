### 首屏、一屏、二屏
首屏：一个页面第一次渲染完成的样子，通常是骨架屏。<br />一屏：app中不需要上下滑动我们见到的那部分屏幕。<br />二屏：app中需要向下滑动才能见到的那部分屏幕。

### 用户感知时间指标
TTI(可交互时间)：页面加载到可以与用户进行交互的时间，也就是从上一个页面点击跳转按钮，到下一个页面一瓶加载完毕的时间。
