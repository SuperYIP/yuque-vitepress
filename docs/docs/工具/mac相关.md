### mac命令行打开文件
```shell
open 文件名
```
此命令会使用mac的默认文本编辑器打开文件
### 右键文件夹 -> 快速操作 -> 通过指定软件打开
### mac电脑查看IP地址
```javascript
ifconfig | grep "inet" | grep "100"
```
